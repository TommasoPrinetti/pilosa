## Workspace Health Matrix

Generate a Unicode status matrix in the workspace index to show multi-dimensional health across groups.

### Status Matrix Rendering

```
For each cell, assign status based on data:
  ✓ = all checks passed
  ⚠ = minor issues or warnings
  ✗ = failures or missing
  ○ = not yet checked
  ◉ = currently processing
```

### Matrix Metrics

| Row | Check | How to Determine |
|---|---|---|
| **Maps** | Group has required maps | Check if group map exists in maps/ |
| **Links** | Wikilinks are valid | Grep for broken [[wikilinks]] |
| **Fresh** | Files are not stale | Compare updated: date against stale_after_days |
| **Dict** | Terms are defined | Check if group terms exist in dictionary.md |

### Dashboard Format

```
┌─ Workspace Health ──────────────────────────────────────────────┐
│ Group    A    B    C    D    E    F                             │
│ Maps     ✓    ✓    ⚠    ✓    ✓    ✗                            │
│ Links    ✓    ✓    ✓    ✓    ⚠    ✓                            │
│ Fresh    ✓    ✓    ✓    ✓    ✓    ✓                            │
│ Dict     ✓    ✓    ✓    ○    ✓    ✓                            │
└─────────────────────────────────────────────────────────────────┘
```

### Status Thresholds

| Overall Score | Status | Action |
|---|---|---|
| All ✓ | ✓ healthy | No action needed |
| Mostly ✓ with few ⚠ | ⚠ minor issues | Review warnings |
| Many ⚠ or any ✗ | ✗ attention needed | Run janitor cleanup |
| All ○ | ○ not started | Run startup protocol |
