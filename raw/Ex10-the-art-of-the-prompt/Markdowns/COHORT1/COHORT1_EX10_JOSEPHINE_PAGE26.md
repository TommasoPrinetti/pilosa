# Ex10 - The Art of the Prompt

**First impressions**
Take some notes during the presentations, paying attention to
what the prompting techniques evoke in you. Are you surprised,
skeptical, curious to try them? What catches your attention?

Plan & Solve: -> for calculations -> propose a plan & then solve
-> write one prompt only
-> Data analysis but can be helpful for reasoning context
emotional prompt: emotional intelligence crucial to human responses
-> "This is very important for my career" = just add on prompt

System 2 Attention: -> regenerate context without irrelevant data
-> produce final answer based on it
-> good in long text prompts

Role Playing: role specific knowledge -> imagine being a researcher...
-> personalizing answers, gives context on what style of
sources are more important

Reverse - Chain of Thought: for arithmetic problems
-> put information into the right order

Rephrase & respond: aim to avoid misunderstanding, often implicits
involved
-> make machine rephrase & respond to see if
it understood correctly
-> make implicit things explicit
-> can be used for multiple contexts
-> "plug in" at the end of prompt

Step-back prompting: 2 step -> borrowed from how human
mind works -> helps with abstraction
-> good for high context - complex problems

Chain of Verification: -> asks to verify these responses given
-> should make the LLM hallucinate less
-> for search tasks
-> 2 step process -> very lengthy

Self-Refine: -> more like a conversation -> about how to improve
the answer, could be used anywhere, time consuming

Self-Ask: add self-ask to prompt, similar to CoT
-> more for info retrieval, search...
-> are there further -- needed
