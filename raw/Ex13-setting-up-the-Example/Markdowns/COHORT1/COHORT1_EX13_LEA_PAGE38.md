# Ex14a - Mapping Exemplary Patterns

Instructions
You will provide context for the tasks your chosen exemplary work represents. You will assess each task against multiple practical criteria, drawing a radar graph to help you reflect on its relevance, significance, and role in your professional practice.

| CATEGORY | DESCRIPTION |
| :--- | :--- |
| Frequency | How often is this task performed in your profession? (1: exceptional, 5: routine) |
| Professional Stakes | How significant are the consequences of success or failure in this task for your professional reputation? (1: Low impact, 5: Critical) |
| Expertise Required | Does the task require specialized knowledge or skills? (1: Basic knowledge, 5: High specialization) |
| Effort and Resources | How resource-intensive is the task (time, effort, tools)? (1: Minimal effort, 5: Significant investment) |
| Collaboration | How collaborative is this task? (1: Independent, 5: Highly collaborative) |

| Professional Pride | How much does this task contribute to a sense of professional accomplishment? (1: Low pride, 5: High pride) |
