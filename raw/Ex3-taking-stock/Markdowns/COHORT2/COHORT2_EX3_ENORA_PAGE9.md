# Ex3c - L'histoire de votre LLM / 2

Si quelqu'un utilisait uniquement vos conversations avec le
LLM pour comprendre votre travail, quels aspects seraient
négligés ou mal représentés ?

Réfléchissez aux ombres, aux contextes manquants et au travail effectué avant et après
l'utilisation du LLM, qui peuvent échapper à ces conversations.

L'organisation ! Je passe mon temps à organiser
les manifestations de différentes manières :
-> RH : déclarer les heures + personnes sur les actions
-> planning ⊕ feuilles de route
-> logistique : hôtels, voyages
⊕ l'artistique : avoir des discussions avec les
artistes, défendre ces visions, ... le réseautage ...
