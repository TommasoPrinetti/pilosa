# Analytical Questions Checklist

Use these questions to structure the contextual analysis. Not all apply to every query — select the relevant ones.

## Corpus Framing
- What does the project's research vocabulary suggest about this topic?
- Which dictionary concepts connect to the user's question?
- What theoretical frames from context.md might apply?

## Coverage Gaps
- What adjacent topics might the raw corpus contain but a keyword search miss?
- What perspectives are likely underrepresented given the source types?
- What temporal, geographic, or demographic blind spots exist?

## Alternative Interpretations
- How else could the user's question be framed?
- What lateral connections exist between this question and other project themes?
- What would a skeptic or contrarian ask about this same topic?

## Bias Check
- What assumptions does a search-only approach carry?
- Which source types might dominate the results and why?
- What would a broader survey reveal that targeted grep would miss?
